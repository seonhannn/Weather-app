현재 위치를 감지하여 날씨를 출력합니다.

지역 검색 버튼을 클릭하면 Search 페이지로 이동해 도시를 검색할 수 있습니다.

검색한 도시의 날씨 정보를 출력합니다.

현재 위치 날씨 버튼을 클릭하면 Main 페이지로 이동해 현재 위치를 감지하여 날씨를 출력합니다.
